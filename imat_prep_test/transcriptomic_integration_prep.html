Create a thresholding Function

Create a function that takes a numeric vector of mean normalized counts and a low and a high quantile cut off. If the normalized gene count is higher than the high quantile, it is assigned a 1. If the If the normalized gene count is lower than the low quantile, it is assigned a -1. All remaining gene counts are assigned a 0.

thresholding_function = function(expr_vector, low_q, high_q) {
  
  low  = quantile(expr_vector, low_q)
  high = quantile(expr_vector, high_q)
  
  ifelse(expr_vector > high, 1,
         ifelse(expr_vector < low, -1, 0))
}

Create a Comparison Function

Create a function to compare how many genes are high and low in each condition.

compare_thresholds = function(mixed_vec, stm_vec) {
  # extract the names of the high genes in the mixed_stm list
  # extract the names of the low genes in the mixed_stm list
  mixed_high = names(mixed_vec)[mixed_vec == 1]
  mixed_low  = names(mixed_vec)[mixed_vec == -1]
  
  # extract the names of the high genes in the stm list
  # extract the names of the low genes in the stm list  
  stm_high = names(stm_vec)[stm_vec == 1]
  stm_low = names(stm_vec)[stm_vec == -1]
 
  # For high genes: 
  # which genes are high in both sets
  # which genes are high only in the mixed set
  # which genes are high only in the stm set 
  list(
    high = list(
      overlap = intersect(mixed_high, stm_high),
      mixed_only = setdiff(mixed_high, stm_high),
      stm_only = setdiff(stm_high, mixed_high)
    ),
      # For low genes: 
  # which genes are low in both sets
  # which genes are low only in the mixed set
  # which genes are low only in the stm set 
    low = list(
      overlap = intersect(mixed_low, stm_low),
      mixed_only = setdiff(mixed_low, stm_low),
      stm_only = setdiff(stm_low, mixed_low)
    )
  )
}
